# Παραμυθούπολη — paramythoupoli.gr

Μη κερδοσκοπική ψηφιακή βιβλιοθήκη παραμυθιών και μύθων για παιδιά.
Περιεχόμενο υπό **CC BY-NC-SA 4.0**.

## Πώς δημοσιεύεις μια ιστορία

1. Πήγαινε στο **paramythoupoli.gr/admin**
2. «Νέα ιστορία» → συμπλήρωσε τη φόρμα → ανέβασε **μία** φωτογραφία
3. Πάτα **Δημοσίευση**

Όλα τα υπόλοιπα γίνονται μόνα τους:

| Παράγεται αυτόματα | Από |
|---|---|
| Σελίδα ιστορίας (HTML, schema, breadcrumb) | Eleventy |
| Κάρτα στην αρχική & στην κατηγορία | Eleventy |
| Μετρητές, ItemList, sitemap.xml, llms.txt | Eleventy |
| «Διάβασε επίσης» — **χωρίς ορφανές ιστορίες** | Eleventy |
| 4 εκδοχές εικόνας (jpg/webp × μεγάλη/μικρή) | GitHub Actions |
| PDF (A5) και EPUB | GitHub Actions |

## Δομή

```
src/
├── stories/*.md        ← ΤΑ ΔΕΔΟΜΕΝΑ. Μία ιστορία = ένα αρχείο.
├── _data/
│   ├── site.js         ← κατηγορίες, σταθερές
│   └── categories.js   ← κείμενα & SEO κατηγοριών
├── _includes/layouts/  ← templates
├── index.njk · category.njk · sitemap.njk · llms.njk
├── static/             ← terms, afigites, favicons, robots.txt, _headers
└── assets/
    ├── uploads/        ← οι πρωτότυπες φωτογραφίες που ανεβάζεις
    ├── img/            ← παράγονται αυτόματα
    └── downloads/      ← PDF + EPUB, παράγονται αυτόματα
admin/                  ← το CMS (η φόρμα)
scripts/build_assets.py ← εικόνες + PDF + EPUB
```

## Σύνταξη μέσα στο κείμενο

- `## Τίτλος` → κεφαλίδα ενότητας
- `---` → λουλούδι-διαχωριστής (με την εικόνα εξωφύλλου)
- `![περιγραφή](/assets/img/x.jpg "λεζάντα")` → εικόνα μέσα στην ιστορία
- `![ΛΟΥΛΟΥΔΙ](/assets/img/x.jpg)` → διαχωριστής με **άλλη** εικόνα
- `<!-- ΔΙΔΑΓΜΑ -->` → εδώ μπαίνει το πλαίσιο του διδάγματος (αλλιώς πάει στο τέλος)
- Το **αρχικό μεγάλο γράμμα** μπαίνει αυτόματα στην πρώτη παράγραφο

## Κανόνες

- Το ήθος βγαίνει από τη **δράση των χαρακτήρων**. Το «Το δίδαγμα» και το «Για συζήτηση»
  είναι προαιρετικά· οι νεότερες ιστορίες δεν τα χρησιμοποιούν.
- Οι στιλιστικές επιλογές του Γ. Σκλαβούνου (κεφαλαίες Έννοιες, λαϊκοί τύποι) **δεν
  «διορθώνονται» ποτέ**.
- Δομικές αλλαγές και νέο πρωτότυπο περιεχόμενο θέλουν έγκριση Γ. Σκλαβούνου.
- Τίτλος Google ≤ 60 χαρακτήρες, περιγραφή ≤ 158. Το CMS το ελέγχει.

## Τοπικά (προαιρετικά)

```bash
npm install
npm start                          # προεπισκόπηση στο localhost:8080
python3 scripts/build_assets.py    # εικόνες + PDF + EPUB
```

## Τεχνικά

Eleventy 3 · Decap CMS · Netlify (auto-deploy από `main`) · GitHub Actions.
Γραμματοσειρές self-hosted. Χωρίς frameworks, χωρίς βάση δεδομένων, χωρίς server.
